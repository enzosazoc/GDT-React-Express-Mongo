# Gestor de Tareas
Aplicación básica para gestionar tareas.

<br>

## Características de la aplicación
---
- Cliente Servidor
- MVC
- CRUD
- API REST
- Autenticación con JWT
- Roles de usuario

<br>

## Tecnologías utilizadas
---
- React
- Bootstrap
- Node.js
- Express.js
- MongoDB


<br>

## Dependecias npm
---
Dependencias globales:
- @babel/core@7.12.3
- @babel/node@7.12.6
- nodemon@2.0.6

<br>

## Comandos
---
### **Ejecutar proyecto**
Para iniciar el proyecto:
```
npm start
```
